# Onboarding 视频脚本 (3-5 分钟)

> 目标: 新客户看完能独立完成 "装 → 激活 → 注册 1 号 → 养号启动" 全流程
> 时长: 3-5 分钟
> 拍摄: 用户自己录 (OBS / Camtasia / ScreenRec 均可)
> 语言: 中文 · 粤语或华语均可

---

## 总览

| Scene | 时长 | 内容 |
|---|---|---|
| 1 | 30s | 下载 + SmartScreen 处理 |
| 2 | 30s | License 激活 |

## Scene 2 · License 激活 (30s)

## Scene 3 · 配代理 + (可选) API Key (60s)

### 画面

**Part A · 代理** (30s):
- 导航 → 代理管理 tab
- 点 "添加代理" · 弹窗
- 填 SOCKS5 · host · port · 账密 (代理商给)
- 点 "测试连接" → 绿勾 "已连接 · 延迟 120ms"
- 点 "保存"
- 列表新增一条

**Part B · AI API (可选)** (30s):
- 导航 → AI 配置 tab
- 4 列 provider 展示 · 全关状态
- 强调: "这一步可跳过 · 产品照样能跑"
- 演示 · 开 DeepSeek · 粘 Key · 点 "测试" → 绿勾
- 省略其他 provider

### 旁白
> "接下来配代理 · 强烈建议配 · 尤其打算跑多个号.
> 加代理 → 填代理商给你的 IP / 端口 / 账密 → 测试连接 → 保存.
> 住宅代理建议 1 个 IP 绑 3 到 5 个号 · 超过 5 个容易被 WhatsApp 关联封号.
>
> 然后是 AI 配置 · 这一步完全可选.
> 不配 AI · 产品照样跑 · 只是文本发的是剧本原文 · 没有个性化改写.
> 推荐用 DeepSeek · 便宜 · 国内也能通. 每月 3-5 美金搞定全部号的 AI 文本.
> Replicate API 用来生成头像 · 大概每月 5 美金 · 不配就用我们预置的图.
> 这些都是可选 · 按预算决定."

### 截图需求
- 代理管理 tab · 添加表单
- 测试连接结果 "已连接 · 120ms"
- AI 配置 tab · DeepSeek 开关 + Key 输入框

## Scene 4 · 注册第一个 WhatsApp (90s)

### 画面
- 导航 → Accounts (槽位管理) tab
- 目前空 · 只有 "Slot 1 · empty" ~ Slot N
- 点某个 empty slot → "绑定账号" 弹窗
- 选代理 (下拉刚才配的代理组)
- 点 "启动注册"
- 填手机号 `60168160836`
- 选 "SMS" · 点 "发送验证码"
- 切到手机实拍 · 展示 SMS 收到 6 位码
- 切回 PC · 输入 6 位码
- 转圈 ~5 秒
- 成功 · 槽位状态变绿 · 显示号码 + "在线" · Phase 0

### 旁白
> "注册第一个 WhatsApp 账号 · 这是整个产品最核心的一步.
> 进 Accounts 页 · 选一个空槽位 · 点绑定账号.
> 分配刚才配的代理 · 点启动注册.
>
> 输入你的马来西亚手机号 · 格式 60 加号码 · 不要加加号.
> SIM 卡必须是全新的 · 或者至少 30 天没用过 WhatsApp.
> 建议选 SMS 接码 · 如果收不到再换 voice call.
>
> 点发送验证码 · 拿起手机等短信 · 收到后输入 6 位码.
>
> 成功后 · 槽位卡变绿 · 显示号码在线 · Phase 等于 0 · 意思是进入孵化期.
> 孵化期 WhatsApp 只接消息 · 不主动发 · 这是防封号的关键."

### 截图需求
- Accounts 页 · 空槽位网格
- 绑定账号弹窗 · 填手机号
- 手机 SMS 截图 · 6 位验证码
- 注册成功 · 槽位绿色状态

## 拍摄注意

### 画面
- 1920×1080 分辨率 · 桌面整洁 (关闭无关通知)
- WAhubX 窗口最大化 · 字体不要太小
- 手机实拍环节 · 竖屏 · 别乱晃

### 声音
- 外置麦克风比笔电内置好 10 倍 · 推荐入门款 $20-50
- 录完降噪处理 (Audacity 免费 · 1 键)

### 剪辑
- 每段之间加 0.3s 淡入淡出
- 光标高亮 (录屏软件一般有选项)
- 敏感信息 (License Key / 真手机号) 打马赛克或用测试数据

### 开源工具
- 录屏: OBS Studio (免费)
- 剪辑: DaVinci Resolve 免费版 / Shotcut / 剪映
- 马赛克: 剪辑工具都内置

# Pilot 客户招募包

> 目的: V1.0 发布前找 3-5 家 pilot 客户 · 2-4 周试跑 · 收反馈调产品
> 受众: 销售 / 创始人自己跑业务发展

---

## 1. 理想客户画像

### 硬需求

客户应同时满足:
- **马来华人** · 或在马来西亚运营华人 / 中文市场
- **已经在用 WhatsApp 做私域** (非完全小白 · 有业务闭环)
- **痛点: 单号容量不够** · 希望扩到 3-10 号
- **有基础 IT 能力** · 或配得起 1 个 "运维小弟" (装机 / 处理 QR / 看 Dashboard)

### 软需求 (加分项)

- 客单价 RM 100+ (能覆盖 License + 代理 + API 成本)
- 团队 2-5 人 · 非个人创业者 (团队化意味着对稳定性要求高)
- 愿意每周 30 min 反馈 · 非 "装完失联" 客户
- 对隐私/数据敏感 · 这是我们本地部署的优势

### 排除

- **纯小白** · 连 WhatsApp Business 都没用过的
- **大规模营销号** (100+ 号 · 纯批量群发) · 我们 V1 slot 上限 Enterprise 也就 50
- **跨境电商依赖 Shopee/Lazada API 的** · 我们不是他们的补充
- **法律灰产** (赌博 / 成人 / 诈骗) · 一律不做

## 2. 目标行业

### 第一优先 (客单价稳 · 复购强)

- **马华电商 · 微商** · 美妆 / 保健品 / 食品 / 小家电
- **美容院 · 美发** · 预约提醒 + 复购 + 老客 VIP
- **烘焙工作室 · 私厨** · 预订 + 团购 + 复购
- **私人教练 · 瑜伽馆** · 课程通知 + 会员

### 第二优先 (可做但门槛高)

- **保险代理** · 合规性要求高 · ToS 风险大 · 需告知客户自担
- **房产中介** · 长周期单 · WhatsApp 只是入口
- **教育培训** · 家长群运营

### 第三优先 (试探性)

- **中小企业主** · 客户管理 + 员工通知
- **餐饮连锁** · 会员 + 促销

## 3. 招募渠道

### 最有效 (已知客户)

- **现有客户关系链** · 让 1 个用户介绍 3 个
- **你自己的 WhatsApp 群** · 如果你有行业群

### 中等 (需投时间)

- **Facebook 群**:
  - "马来西亚电商卖家交流" · 几万人大群
  - "Klang Valley 微商联盟"
  - 搜 `Malaysian entrepreneur` / `马华电商`
- **行业微信群**:
  - 马来西亚华商会 / 同乡会类
  - 找 "华人在马来" 类群
- **行业展会 · 小型 meetup**:
  - JomExpo / Malaysia eCommerce Expo
  - KL / PJ 本地 startup meetup

- 冷邮件 (开封率低)
- LinkedIn (马来华人不重用 LinkedIn)
- 付费广告 (V1 pilot 阶段别花)

## 5. Pilot 合同模板 (**需法律审阅**)

### ⚠ 重要

以下是**非法律专业起草**的条款框架. **必须经律师审阅** · 特别是马来西亚本地律师 (熟悉 PDPA + 消费者法).

本产品条款 · 价格 · 服务范围属商务决策 · 触发 HALT 条件 #1 · 用户自定.

### 条款框架 (草稿 · 勿直接用)

```
WAhubX Pilot 测试协议

甲方: [客户公司名]
乙方: [你公司名]

1. 试用期
   - 起: [日期]
   - 止: [日期 · 建议 2-4 周]
   - 免费延期可协商

2. 使用范围
   - 甲方获得 1 个 WAhubX License (套餐 [Basic/Pro/Enterprise])
   - 仅限 [X] 台指定机器部署
   - 仅用于甲方自有业务 · 不得转售或对外分发

3. 费用
   - Pilot 期内 · 乙方不收取 License 费
   - 代理 / AI API 等第三方服务 · 甲方自费 (见 DEPLOYMENT-MODES.md)
   - Pilot 结束后 · 甲方可购买常规 License (价格 [RM X/年])

4. 反馈义务
   - 甲方每周填一次反馈问卷 (5 问 · 10 分钟)
   - 每 2 周一次 30 分钟 check-in 电话
   - 发现 bug 按 FEEDBACK-COLLECTION.md 格式报告

5. 免责
   - 本产品仅为工具 · 甲方对使用产生的 WhatsApp 账号封禁 / 业务损失负全责
   - WhatsApp ToS 风险甲方自担 · 乙方不保证账号寿命
   - 乙方尽力修复 bug · 但不保证 100% 可用率
   - V1 已知限制见 KNOWN-LIMITATIONS-V1.md · 甲方确认已阅读

6. 数据 / 隐私
   - WAhubX 本地部署 · 乙方不主动获取甲方 WhatsApp 数据
   - 甲方提交反馈时自愿提供的 log / 截图 · 视为授权乙方用于产品改进
   - 遵守 PDPA 2010 · 双方均负保密义务

7. 终止
   - 任何一方 7 天书面通知可终止
   - 终止时 · 甲方删除所有 WAhubX 软件 · 乙方 revoke License Key

8. 争议
   - 适用马来西亚法律
   - KL 仲裁

甲方签字: __________  日期: __________
乙方签字: __________  日期: __________
```

## 6. 首批 Pilot 建议 3-5 家

### 选多样性 (覆盖不同 segment)

| 类型 | 规模 | 作用 |
|---|---|---|
| 1 · 大 (PJ/KL) | 10+ 号 · 团队 5+ 人 · 月营收 RM 100k+ | 验 Enterprise 套餐 · 压力测试 |
| 2 · 中 (二三线城市) | 3-5 号 · 团队 2-3 人 · 马华电商 | 主 target · Pro 套餐 |
| 3 · 小 (个人/夫妻店) | 1-2 号 · 个体 | Basic 套餐 · 易上手 |
| 4 · 技术型 (愿意自配 GPU) | 3-5 号 + ComfyUI 本地 | 验 Mode B1 ComfyUI 路径 |
| 5 · 纯免费模式 | 1-2 号 · 预算 0 | 验 Mode A · 产品兜底能力 |

### 每家 pilot 安排

- 装机日 · 60 分钟 · 远程协助
- 第 1 周末 · check-in 30 分钟
- 第 2 周末 · check-in 30 分钟
- 第 3-4 周 · 邮件反馈 + 必要时电话
- 结束: 收总结问卷 · 决定是否转正式付费

## 7. 报价梯度 (Pilot 版)

### Free Pilot (零门槛)

- License: 免费 30 天
- 代理 / AI: 完全可选 · 客户零额外付费也能跑
- 适合: Segment #3 个体 / 技术型 #4 ComfyUI 自建
- 结束后: 付 License 基础版

### Standard Pilot

- License: 免费 30 天
- 客户自购: 代理 ~$40-60/月 + DeepSeek ~$5/月 + Replicate ~$5/月 = **~$50-70/月运营成本**
- 适合: Segment #1, #2 主 target
- 结束后: 付 License + 继续运营成本

### Premium Pilot (推给客单价高的)

- License: 免费 30 天
- 客户自购: ~$150-300/月 (包 ElevenLabs + Claude + flux-pro)
- 适合: Segment #1 大客户 · 单号月产出 $500+
- 结束后: 付 License + 继续高端运营

### **不强推付费**

让客户选 tier. Free pilot 也是合法 option. 主要目的是**收反馈 + 建立 testimonial** · 不是 pilot 期赚钱.

## 8. Pilot 期管理节奏

### Week 0 (装机)

- 远程 Zoom/TeamViewer · 一起装
- 跑通注册 1 号
- 启动养号
- 设置每周 check-in 时间

### Week 1

- 第 3 天 · 短信检查 "号还在线吗?"
- 第 7 天 · 30 min 通话
  - 遇到什么问题?
  - 注册流程顺不顺?
  - 有想加的功能吗?

### Week 2

- 30 min 通话
- 看 Dashboard / Health 数据
- 养号日历跑完了吗

### Week 3-4

- 邮件 check-in
- 每天收 bug report (FEEDBACK-COLLECTION.md 格式)

### Week 4 收官

- 60 min 深度访谈
- 决定: 转付费 or 感谢离开
